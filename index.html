What is PPLV?

PPLV[1] (Principles of programming, Logic and Verification) is a research area focused on Computational Logic and it's applications to other areas.
Research Projects in this area often either seek to develop new computational logic theories or to use them to create rigorous mathematical models that accurately describe and predict physical systems.

What is Computational Logic?

As PPLV is a research area based around the core concept of Computational Logic[2], before moving forward, it's probably best to first explain what logic is...


A Brief Introduction to Logic

Note: For the purpose of brevity, this overview is very rough - if you are interested, check the reference for links that can explain this topic in greater detail

Logic, in the mathematical sense is the study of developing rules and operations to describe concepts. In Computer Science, Logic is specifically used to describe Computational Algorithms.

An analogous example which may illustrate the concept more clearly is Boolean Logic and its relationship with circuits.

In Boolean Logic, values are either True (1) or False (0) and there exist operators (AND, OR, XOR etc.) which return True or False, depending on their input. As these exact operations and values are representable using circuits, we could describe Boolean Logic as a theoretical description of circuits.


Computational Logic encompasses Boolean Logic, but also includes other types of Logic such as Separation Logic and Hoare Logic, which describe computers at a higher level of abstraction (as in describing the programs on the computer rather than the bits that make it up).

Incidentally, one of the core developments in Computational Logic, which is now used throughout the world to verify imperative programs before release, Separation Logic, was developed by a Researcher at UCL's PPLV department - Peter O'Hearn.


So, what does PPLV do?

It is a research field which rigorously analyses and develops on the systems underlying all modern technologies we currently use. This means that it's findings have impacts in a wide variety of fields - from electrical (circuit logic) to mechanical (self-driving cars) and even to financial fields (Productive Security).

PPLV achieves this by focusing on studying logic – the fundamental language in which all systems can be described. The name Principles of Programming, Logic and Verification hints at the field's main sub-areas.

However, you would be missing the underlying aims of the area if you tried to split it into sub-areas - PPLV's main focus on Logic means that any single research project inevitably involves combining various amounts of knowledge from the other sub-areas. Hence any researcher studying in the field of PPLV knows the underlying Principles of Programming (to understand the actions that a logical model must be able to recreate), Verification (to ensure the accuracy of their model) and Logic (to actually produce the model).

It is a long rigorous process requiring large amount of research, time and skill.
